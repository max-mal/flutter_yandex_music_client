# YaMusic

Yandex Music client written on flutter

Supported: 
- playlists
- feed
- search
- likes / dislikes

Not supported (yet?) :
- radio
- collections
- songs lyrics


Authorization:
- click request code button
- log into your yandex account in browser
- after you being redirected to yandex music site copy code parameter from URL to app
- you are authorized!
